# Pizza-Sales-Data-Analysis | SQL + Excel - Dashboard
SQL + Excel Power Query + Power Pivot Tables + Interactive Dashboard

This project analyzes a Pizza Sales dataset to generate business insights using SQL and Excel.

The dataset was first imported into SQL Server, where SQL queries were written to 
calculate key business metrics such as revenue, total orders, and average order value. 
These metrics were used to validate and cross-check the results generated from Excel Pivot Tables.

The same dataset was then imported into Excel Power Query and used to build Pivot Tables and 
an Interactive Dashboard to visualize sales trends and business insights.

This project demonstrates how SQL and Excel can be used together for data validation and business reporting.

## Tools Used
- SQL Server – Writing SQL queries for KPI calculations and business analysis
- Excel Power Query – Data import and transformation
- Excel Pivot Tables – Data aggregation and summarization
- Excel Dashboard – Interactive data visualization
  
## Key Performance Indicators (Calculated in SQL)
The following KPIs were calculated using SQL queries and used to cross-check the results from Pivot Tables:

 | KPI | Value |
|-----|------|
| Total Revenue | $817,860 |
| Total Orders | 21,350 |
| Total Pizzas Sold | 49,574 |
| Average Order Value | $38.31 |
| Average Pizzas Per Order | 2.32 |

## Pivot Table Analysis
Pivot Tables were created in Excel to analyze the dataset and generate insights such as:

- **Daily Trend of Orders** – Orders are highest on Friday
- **Hourly Trend of Orders** – Peak ordering hours occur between **12 PM – 1 PM** and **5 PM – 7 PM**
- **Sales by Pizza Category** – Classic category contributes the highest revenue
- **Sales by Pizza Size** – Large pizzas account for the highest share of sales
